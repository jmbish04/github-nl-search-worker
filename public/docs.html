GitHub NL Search Worker

This Worker exposes REST, WebSocket, and MCP endpoints for orchestrating GitHub repository discovery and Cloudflare scaffold generation.

Download the OpenAPI specification at /openapi.json.

Quick Start

  1. Create a session via POST /api/sessions.
  2. Start a search with POST /api/sessions/{session_id}/search?wait=true.
  3. List results or generate scaffolds with POST /api/scaffolds.
